# Structure for docs
```
docs/
├── README.md                     # 文档入口，介绍文档结构和主要文档链接
│
├── 01_project_overview/
│   ├── REQUIREMENTS.md           # 需求文档 (功能性、非功能性需求)
│   └── GLOSSARY.md               # 术语表 (项目特定术语定义)
│
├── 02_architecture_and_design/
│   ├── ARCHITECTURE.md           # 系统架构图和详细说明
│   ├── SYSTEM_DESIGN_DECISIONS.md # 或 ADRs/ (架构决策记录)
│   ├── DATABASE_SCHEMA.md        # 数据库设计 (表结构、关系、ER图)
│   └── API_DESIGN.md             # API 设计规范 (RESTful风格、版本控制、认证授权)
│
├── 03_development/
│   ├── DEVELOPMENT_ENVIRONMENT_SETUP.md # 开发环境搭建指南 (软件、工具、配置)
│   ├── BUILD_AND_RUN.md          # 项目构建和本地运行指南
│   ├── LIBRARIES_AND_FRAMEWORKS.md # 使用的主要库和框架说明 (及选择理由)
│   └── ASPIRE_INTEGRATION.md     # .NET Aspire 在项目中的具体使用和配置说明
│
├── 04_api_reference/
│   ├── (Generated by Swagger/OpenAPI) # 通常是自动生成的 API 文档
│   └── API_USAGE_EXAMPLES.md     # API 使用示例和场景
│
├── 05_deployment_and_operations/
│   ├── DEPLOYMENT_TO_AZURE.md    # Azure 部署步骤 (App Service, AKS, DB, Blob等)
│   ├── CI_CD_PIPELINE.md         # CI/CD 流程说明 (Azure DevOps/GitHub Actions)
│   ├── MONITORING_AND_LOGGING.md # 监控和日志策略 (Azure Monitor 配置)
│   ├── BACKUP_AND_RECOVERY.md    # 数据备份和恢复计划
│   └── TROUBLESHOOTING_OPERATIONS.md # 线上常见问题排查手册
│
└── 06_testing/
│   ├── TESTING_STRATEGY.md       # 测试策略 (单元、集成、E2E、性能测试)
│   ├── TEST_PLAN.md              # 测试计划 (主要测试用例、测试环境)
│   └── HOW_TO_RUN_TESTS.md       # 如何运行各类测试
```
